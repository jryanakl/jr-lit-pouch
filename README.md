## Getting Started
```
git clone https://github.com/jryanakl/jr-lit-pouch.git
cd jr-lit-pouch
npm install
npm run build
npm run dev
```

## To view this application
The application will open to [http://localhost:5173/](http://localhost:5173/) in your browser.
